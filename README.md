# Multimodal Neural Network from Scratch

> **Мультимодальная нейронная сеть, объединяющая CNN, RNN и полносвязные слои для решения задачи бинарной классификации.**

## 📖 О проекте

**Multimodal Neural Network from Scratch** — исследовательский проект, посвященный разработке мультимодальной архитектуры глубокого обучения, способной одновременно обрабатывать несколько различных типов входных данных.

В отличие от классических моделей, работающих только с одним видом информации, данная архитектура объединяет несколько специализированных нейронных сетей:

* **Convolutional Neural Network (CNN)** — для извлечения пространственных признаков;
* **Recurrent Neural Network (RNN)** — для анализа последовательностей и временных зависимостей;
* **Fully Connected Network (Dense)** — для обработки табличных признаков и объединения информации.

Полученные представления объединяются в единое пространство признаков и используются для решения задачи **бинарной классификации**.

Проект демонстрирует построение сложной нейронной архитектуры практически с нуля и показывает принципы организации мультимодальных моделей в современных системах искусственного интеллекта.

---

# 🎯 Основная идея

Каждый тип данных содержит собственную информацию.

Вместо использования одной универсальной модели проект применяет специализированную архитектуру для каждой модальности, после чего объединяет полученные признаки перед финальной классификацией.

Такой подход позволяет использовать сильные стороны каждого типа нейронной сети и получать более информативное внутреннее представление данных.

```text

        ┌───────────────┼────────────────┐
        │               │                │
        ▼               ▼                ▼
```

# ⚙️ Pipeline обучения

Процесс работы модели состоит из нескольких этапов.

### 1. Подготовка данных

Исходные данные разделяются на несколько модальностей в зависимости от их типа.

Каждая группа признаков направляется в соответствующую специализированную нейронную сеть.

---

### 2. Извлечение признаков

Каждая ветвь модели независимо извлекает информативные представления:

* **CNN** анализирует пространственную структуру данных;
* **RNN** выявляет временные зависимости;
* **Dense** преобразует табличные признаки в компактное представление.

---

### 3. Feature Fusion

Полученные скрытые представления объединяются в единый вектор признаков.

Именно этот этап позволяет модели учитывать информацию сразу из нескольких источников одновременно.

---

### 4. Финальная классификация

После объединения признаков они проходят через несколько полносвязных слоев, формируя вероятность принадлежности объекта к одному из двух классов.

---

# 💡 Почему мультимодальная архитектура?

Использование нескольких специализированных нейронных сетей позволяет:

* учитывать различные типы информации одновременно;
* извлекать более информативные признаки;
* повысить качество классификации;
* строить архитектуры, близкие к современным промышленным решениям.

Подобный подход широко применяется в компьютерном зрении, обработке естественного языка, медицинской диагностике и интеллектуальных рекомендательных системах.
