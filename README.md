# Multimodal-Neural-Network-from-scratch
В репозитории представлена мультимодальная нейронная сеть (CNN + RNN + Dense), на вход которой подаются изображения формата (1000, 1, 384, 384), то есть (batch_size, channels, height, width) и числовые последовательные данные с 27 признаками в формате (1000, 1, 27), то есть (batch_size, sequence_length, num_features)
